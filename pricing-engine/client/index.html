    Spec Groups are the options a customer picks — Material, Size, Printing, Corners, etc.

    Each group has options. An option can be one of three types:

    Base (curve) — the primary material/stock. Uses a power curve to calculate cost: cost = setup_cost + per_unit_cost × qty^scale_factor Only your first spec group (usually Material) should have Base options. The base already includes the default print side (front only).

    Modifier — a flat per-unit add-on: extra = cost_modifier × qty. Use this for anything that adds a fixed cost per piece — rounded corners ($0.005/unit), double-sided printing ($0.008/unit), larger size ($0.010/unit). A $0 modifier is valid — it means "no extra cost" but keeps the option explicit in the table (e.g. Front Only = $0, Straight Corners = $0).

    Surcharge — a percentage of the base cost: extra = base_cost × pct%. Use this for job-level premiums like rush fees or special handling that scale with the job value, not the unit count.

    What is Scale Factor?

    Scale factor controls how fast your cost grows as quantity increases. It creates the bulk discount curve — higher quantities cost more in total, but less per unit.

    print cost = setup_cost + per_unit_cost × qty scale_factor

    The lower the scale factor, the steeper the bulk discount:

    Scale FactorBehaviourGood for
    0.50Steep discount — cost barely grows with qtyDigital / low marginal cost
    0.65Moderate discount — typical print productBusiness cards, flyers
    0.80Mild discount — cost grows close to linearlyPremium / labour-heavy items
    1.00No discount — fully linear costCustom / one-off work
